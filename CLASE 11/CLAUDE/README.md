# 15 Free Claude Skills

A pack of 15 ready-to-use Claude skills covering common use cases: writing, marketing, sales, productivity, analysis, and creator workflows.

## What's a Claude skill?

A skill is a folder with a `SKILL.md` file. When you install it, Claude loads the instructions inside whenever your request matches what the skill is designed for. Think of it as codified expertise — you install once, then Claude behaves like it's read the playbook every time.

## What's inside this ZIP

- **15 skill folders** (e.g. `blog-post-writer/`) — the editable source. Open any `SKILL.md` and customize it.
- **`installable-skill-files/`** — the same 15 skills pre-packaged as `.skill` files, ready to upload.

## How to install (easy)

Use the pre-packaged `.skill` files:

1. Open Claude.ai
2. Go to **Settings** → **Capabilities** → **Skills**
3. Upload any `.skill` file from `installable-skill-files/`
4. Done — Claude will use that skill whenever your request matches it

Install only the ones you want, or upload all 15.

## How to install (Claude Code / API)

If you're using Claude Code or building on the API:

1. Drop the skill folder (e.g. `blog-post-writer/`) into your skills directory
2. Claude will pick it up automatically

## How to use

Just ask Claude normally. For example:

- "Write a LinkedIn post about my new product launch" → triggers `social-media-post-writer`
- "Can you analyze this competitor's pricing page?" → triggers `competitor-teardown`
- "Help me do a weekly review" → triggers `weekly-review`

You don't need to call the skill by name. Claude reads the description of each installed skill and picks the right one automatically.

## Customizing

Each `SKILL.md` is plain markdown. Open it, edit it, make it match your voice or workflow. Common tweaks:

- Add your own examples of good output
- Change the default output format
- Add brand-specific voice rules
- Tighten or expand the scope

## Questions?

These skills are designed as starting points. They work well out of the box, but they get better when you tailor them to how you actually work. The more specific your skill, the better Claude performs.
