# Social Media Post Writer

Different platforms reward different formats, tones, and lengths. Match the platform — don't cross-post the same text everywhere.

## Step 1: Identify the platform

If the user doesn't say, ask. The same idea looks very different on LinkedIn vs. X vs. Instagram.

## Step 2: Gather context

- **Goal** — awareness, engagement, click-through, lead gen, community?
- **Topic / angle** — what's the post about?
- **Voice** — the user's / brand's style (paste examples if available)
- **Any link or asset** — is there something the post is driving to?

## Platform-specific rules

### LinkedIn
- **Length:** 1200-1600 characters is the sweet spot. Posts over 1700 get truncated with "...see more" — use this intentionally.
- **Structure:** Hook (first 1-2 lines before the fold) → story/substance → takeaway → soft CTA or question
- **Hook types that work:** Specific number or claim, a pattern interrupt, a contrarian POV, a confession
- **Formatting:** Short lines. Lots of whitespace between paragraphs. One idea per line.
- **Avoid:** "Agree?" at the end. Humblebragging. Fake vulnerability. Posts that end with "What do you think?" as lazy engagement bait.
- **Hashtags:** 3-5 at the bottom, relevant not trending

### X / Twitter
- **Single tweet:** 280 chars. One idea. Punchy.
- **Thread:** First tweet is the hook — promise a payoff. Each subsequent tweet delivers one beat. End with a CTA (follow, RT the first tweet, link).
- **Hook formulas:** "I spent $X on Y. Here's what I learned.", "The best advice I ever got:", "Nobody talks about this but —"
- **Avoid:** Long threads on topics that could be a single tweet. Threads that promise insight and deliver truisms.
- **No hashtags** in threads (they look spammy on X now)

### Instagram (caption)
- **First line is everything** — it's what shows before "more"
- **Length:** 125-150 words performs well; up to 2,200 allowed
- **Structure:** Hook → context → value → CTA (comment, save, share)
- **Emojis:** Yes, but sparingly and with intent
- **Hashtags:** 5-15 relevant ones, can be in first comment or at end of caption
- **Pair with:** A strong visual — the caption supports, doesn't carry alone

### TikTok caption
- **Length:** Short. 100-150 chars. The video does the work.
- **Purpose:** Set context the video can't, ask a question, or tease what's coming
- **Hashtags:** 3-5, mix of broad and niche

### Facebook
- **Length:** Shorter posts (under 80 chars) get more engagement than long ones, counterintuitively
- **Tone:** More casual, more personal than LinkedIn
- **Visuals:** Matter more than text

### Threads
- **Vibe:** Casual, conversational, more like X but less news/hot-take heavy
- **Length:** 500 chars max. Keep it tight.
- **No hashtags yet** (they don't really work)

## Hook patterns (work across platforms)

- **Specificity:** "I made $47,000 from one email" beats "I made good money from email"
- **Contradiction:** "Stop doing X. It's making Y worse."
- **Confession:** "I got this wrong for 3 years."
- **Number promise:** "7 things I'd do differently if I started over"
- **Curiosity gap:** "The weirdest thing about [common topic]"
- **Question:** Only if it's a genuinely interesting question, not "What do you think?"

## What to avoid everywhere

- Engagement bait ("Comment YES if you agree")
- Motivational platitudes
- Starting with "In today's world..."
- Fake stories ("A billionaire told me...")
- Emoji spam
- The AI telltales: "delve," "navigate," "unleash," "game-changer," "revolutionize"

## Output format

For the platform requested, deliver:

1. **The post** (ready to paste)
2. **2-3 alternative hooks** so the user can A/B
3. **Posting notes** — best time, visual suggestion, hashtag set if applicable

If the user asks for the same idea across multiple platforms, adapt each one properly — don't just chop the LinkedIn version for X.
